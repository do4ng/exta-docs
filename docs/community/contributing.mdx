## Project Structure

- [`/packages/`](https://github.com/do4ng/exta/tree/main/packages): Core Packages Directory

## Contribute

1. Fork and clone [repository](https://github.com/do4ng/exta)

```sh
git clone https://github.com/do4ng/exta.git
cd core
```

2. Install dependencies

> This repository uses yarn berry. [yarn](https://yarnpkg.com/) must be installed.

```sh
yarn install
```

3. Make changes
4. Open a [Pull Request](https://github.com/do4ng/exta/pulls)
